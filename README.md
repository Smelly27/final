## Problem
Planning international travel involves managing destinations, dates, currencies, and cultural context. This project provides a centralized, interactive interface to manage itineraries while enhancing the experience through themed travel guidance.

## Features
- Create, edit, and delete travel itineraries
- Currency conversion for international planning
- Destination recommendations based on food preferences
- Rotating explorer-style travel notes
- Fully containerized with Docker

## Technologies Used
- Node.js
- Docker
- GitHub
- GitHub Actions (CI/CD)
- AWS (Infrastructure as Code)

## DevOps Practices
- Git branching and pull requests
- Automated builds
- Container-based deployment
- Infrastructure defined programmatically

This project was built to demonstrate full-stack DevOps principles in a real-world scenario.
